Cloud9 Workspace Manager

Currently Cloud9 IDE open source component does not have any workspace management capabilities which makes IDE use cases very limmited. Purpose of this project is to provide workspace and user management for Cloud9 IDE.

About Errors None of the errors are displayed in user interface. All will fail silently.

About Login Entering invalid username and password does not warn user. It will fail silently.

About Sign up Sign up is not yet implemented. Button will not work. Administrators can create users in users menu.
